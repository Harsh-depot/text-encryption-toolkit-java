# 🔐 Text Encryption Toolkit (Java)

A Java-based command line toolkit that implements classic text encryption and manipulation algorithms.

This project was developed as part of the **AICTE - EduSkills Java Full Stack Virtual Internship**.

---

## 🚀 Features

The toolkit provides the following functionalities:

- Caesar Cipher Encryption
- Caesar Cipher Decryption
- ROT13 Cipher
- Reverse Cipher
- Letter Frequency Counter

These algorithms demonstrate fundamental cryptographic techniques and string manipulation concepts in Java.

## 🛠 Technologies Used

- Java
- Java Standard Library
- Scanner (for input handling)

## 🔑 Algorithms Implemented

### Caesar Cipher
Shifts each alphabetic character by a fixed number of positions.

Example:


Input: Hello
Shift: 3
Output: Khoor


---

### ROT13 Cipher

A special Caesar Cipher with a shift of **13**.


Input: Hello
Output: Uryyb


---

### Reverse Cipher

Reverses the entire string.


Input: Hello
Output: olleH


---

### Letter Frequency Counter

Counts alphabetic characters in text.


Input: Hello123
Output: 5
